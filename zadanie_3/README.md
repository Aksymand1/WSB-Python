# my_awesome_lib

Prosta biblioteka demonstracyjna do laboratorium „Tworzenie własnej biblioteki programistycznej w Pythonie”.

## Funkcjonalności

### data_utils
- `usun_duplikaty(data)` - usuwa duplikaty z listy zachowując kolejność elementów
- `ListProcessor` -  Klasa zawierająca metody do przetwarzania list zawierających liczby całkowite. 

### math_tools
- `suma_cyfr_liczby(number: int)` - zwraca sumę cyfr podanej liczby całkowitej
- `srednia(liczby: list)` - oblicza średnią z elementów listy
- `silnia(n: int)` - oblicza silnię z podanej liczby n
- `fibonacci(n: int)` - zwraca pierwszą znalezioną liczbę Fibonacciego zawierającą n cyfr oraz jej indeks

### text_processing
- `licz_slowa(tekst: str)` – zwraca liczbę słów w podanym tekście
- `snake_to_camel(tekst: str)` – konwertuje tekst z formatu snake_case na CamelCase
